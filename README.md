# Wallet Web App

A modern, responsive Wallet Web App interface built with HTML, TailwindCSS (CDN), and JavaScript.

## Deployment on Render

This project is configured for easy deployment on Render.

1.  Log in to your [Render Dashboard](https://dashboard.render.com/).
2.  Click **New +** and select **Blueprint**.
3.  Connect this repository (`munna1234mm/developermunna54`).
4.  Render will automatically detect the `render.yaml` file and set up your Static Site.
5.  Click **Apply** to deploy.

Your site will be live in seconds!
